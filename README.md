**How to setup**

Download & install Termux from F-Droid - 
https://f-droid.org/en/packages/com.termux

Inside termux, install SSH - [Here is how](https://github.com/YasinSuzon/Termux-Commands/blob/main/To%20install%20SSH.yml)  
  
SSH is only needed to access the android terminal from other devices. Which will be easier to maintain the device & copy-paste the terminal commands.

Use SSH from PC to connect to your android device. Then use the install command.  

  
**After Installation**
- Create 'Movies/Music' folders via Copyparty
- Add those folder as Library on Jellyfin
